# Задание 1

## Задача 1

### Текст задачи
Разработать консольное приложение с дружественным интерфейсом с возможностью выбора 
заданий для работы с «базой данных (БД)», хранящейся в бинарном файле. Перечень полей 
(минимум 5), достаточно полно характеризующих заданную в варианте предметную область, 
предложить самостоятельно (постараться отразить в перечне полей такие, которые требуют разных 
типов данных). Приложение должно выполнять следующие функции:
1. Чтение базы данных из бинарного файла.
2. Просмотр базы данных.
3. Удаление элементов (по ключу).
4. Добавление элементов.
5. Реализация 4 запросов (формулировки запросов придумать самостоятельно). 2 запроса 
должны возвращать перечень, 2 запроса одно значение
### Алгоритм решения
Программа разделена на три логических блока:
1. Класс предметной области (Book)
Содержит приватные поля для хранения данных о книги(Id, Название, Автор, Цена, Год выпуска, Статус в наличии).
Содержит свойства с логикой валидации данных.
Содержит перегруженный метод ToString() для форматированного вывода.
2. Вспомогательный класс (BookHelper)
Содержит методы работы с файлом (Добавление книги, удаление по айди, сохранение в файл, выгрузка из файла).
Содержит LINQ-запросы в виде отдельных методов (сортировка по автору, сортировка по цене, самая старая книга, средняя цена всех книг).
3. Основной цикл программыь(Program)
Консолтное меню для вызова функций.
Отлавливание ошибок при помощи try-catch.
